# Package Express – Shipping Quote Console App

A C# console application built using the .NET Framework that calculates a shipping quote based on package weight and dimensions.

## Requirements Implemented
- Weight limit validation (max 50)
- Dimension limit validation (width + height + length ≤ 50)
- Shipping quote calculation
- User-friendly console prompts
- Fully commented source code

## Technologies Used
- C#
- .NET Framework
- Console Application

## How to Run
1. Open the solution in Visual Studio
2. Build the project
3. Run the console application
